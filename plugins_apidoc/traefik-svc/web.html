Traefik Service

Type: object

Create a traefik service for a given service
Example:

stacks:
- app: my_app_name
  tags:
  - traefik-svc
  vars:
  - traefik_svc_entrypoints: default-http,default-https
  - traefik_svc_port: 8080
  - traefik_svc_group: trafik_prod
